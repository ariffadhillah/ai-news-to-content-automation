# AI Legal Incident Content Engine

An AI-powered content automation system that transforms real-world injury incidents into publication-ready legal education content.

## Overview

This project demonstrates how AI, automation, and data processing can be combined to build a scalable legal content workflow.

The system:

* Ingests incident data
* Scores incidents based on severity and relevance
* Generates legal education content using LLMs
* Produces SEO-ready blog drafts
* Creates FAQs and social media content
* Exports publication-ready Markdown files

## Features

### Incident Scoring Engine

Automatically evaluates incidents using factors such as:

* Fatalities
* Injury severity
* Commercial vehicle involvement
* Child injuries
* DUI allegations

### AI Content Generation

Generates:

* SEO Titles
* Meta Descriptions
* Article Summaries
* Full Blog Articles
* FAQs
* Suggested Internal Links
* Suggested External Citations
* Social Media Posts

### Export Workflow

Automatically exports generated content as Markdown files for editorial review.
